📋 核心流程

1
创建 API Key

首次使用时创建访问凭证，保存返回的 key 用于后续请求。

POST /api/v1/api-keys
{ "name": "AI Agent Bot", "email": "agent@example.com", "agentType": "telegram_bot" }
2
浏览产品

获取所有可购买的冷钱包产品列表（无需认证）。

GET /api/v1/products
3
创建订单

选择产品、数量、支付币种，提供收货信息，获取支付地址。

POST /api/v1/orders
Headers: { "X-API-Key": "fsk_xxxxx..." } { "product_id": "uuid", "quantity": 1, "payment_currency": "BTC", "shipping": { "name": "张三", "phone": "+86 138 0000 0000", "address": "深圳市南山区科技园", "city": "深圳", "province": "广东", "country": "中国", "postal_code": "518000" } }
4
确认支付

用户完成支付后，提交交易哈希确认付款。

POST /api/v1/orders/{order_id}/payment
Headers: { "X-API-Key": "fsk_xxxxx..." } { "tx_hash": "a1b2c3d4e5f6..." }
5
查询订单

随时查询订单状态、物流信息。

GET /api/v1/orders/{order_id}
